Which of the following accurately describes the time complexity of binary search on a sorted array of n elements?

AO(n) — Linear time
BO(log n) — Logarithmic time
CO(n²) — Quadratic time
DO(1) — Constant time
✅ B is correct. Binary search halves the search space each step — max steps = log₂n, hence O(log n).
❌ A describes linear search. ❌ C describes bubble sort (nested loops). ❌ D means no scaling with input — impossible for search.
